Driver Drowsiness Detection

This project aims to reduce accidents caused by drowsy driving by detecting signs of drowsiness in real-time and alerting the driver.

Project Details

  • Built a machine learning model to detect drowsiness using facial landmarks and eye aspect ratio
  • Implemented two models: ZFNet (89.59% accuracy) as primary and CNN (93.96% accuracy) as fallback
  • Connected with Arduino to provide physical alerts (vibration and sound) when drowsiness is detected
  • Optimized for real-time performance with OpenCV for video processing

Technologies Used

Python TensorFlow Keras OpenCV Arduino Computer Vision
